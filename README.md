# Programmers Solving Timer

프로그래머스 문제 풀이 시간을 측정하는 Chrome 확장 프로그램입니다.

> 이 프로젝트는 프로그래머스와 공식적으로 제휴하거나 승인받은 제품이 아닙니다.

## 현재 기능

- 프로그래머스 문제 상세 페이지에서 타이머 표시
- 타이머 시작
- 일시정지
- 이어서 시작
- 초기화
- 중복 시작 방지

## 예정 기능

- 문제별 풀이 시간 저장
- 새로고침 후 타이머 복원
- 정답 결과 감지
- 정답 처리 시 타이머 자동 종료
- 문제 목록에 풀이 시간 표시

## 설치 방법

1. 이 저장소를 내려받습니다.
2. Chrome 주소창에서 `chrome://extensions`로 이동합니다.
3. 개발자 모드를 활성화합니다.
4. `압축해제된 확장 프로그램을 로드합니다`를 선택합니다.
5. 프로젝트 폴더를 선택합니다.
